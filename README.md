Instructions to use this repository:
====================================

Run as root below commands:
1.  emerge layman
2.  layman -o 'https://raw.github.com/tm-calculate/tm-calculate/master/tm-calculate.xml' -f -k -a tm-calculate
3.  Enjoy! (:
